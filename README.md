# CitizenM - Hackaton

## Setup

### Prerequisites

- Node.js - v18.0.0 or newer

**Make sure to install the dependencies:**

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000` + open browser automatically:

```bash
npm run dev
```
